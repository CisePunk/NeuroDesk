# NeuroDesk Frontend

Frontend for NeuroDesk, built with React and Vite.

This interface connects to the Spring Boot backend and provides the main workspace for students, modules, tasks, and dashboard views.

## Tech stack

- React 19
- Vite
- React Router
- Axios
- Custom CSS
- Dark / light mode with localStorage persistence

## Running locally

```bash
npm install
npm run dev
